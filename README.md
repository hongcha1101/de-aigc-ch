# 去AI味 · 中文版

去除中文写作中可预见的 AI 语病。基于 2025–2026 年最新学术研究（40+ 项语言特征指标、多模型对比数据），系统识别和消除 AI 写作痕迹。

## 一句话：你在改什么？

AI 写中文有一个稳定可预测的模板：

```
在这个快速发展的时代，[话题]不仅发挥着越来越重要的作用，
更是扮演着不可或缺的角色。值得注意的是，[话题]不仅……而且……
甚至……。综上所述，[话题]具有深远的意义。
```

这套模板在 AI 文本中反复出现——冒号/破折号密度是人类的 **2.7–3.2 倍** [3]，句长方差只有人类的 **1/3** [3][1]，
"不只是……更是……" 的出现频率是人类的 **3.7 倍** [1][2]。本技能把这些模式系统化，逐层消除。

## 快速示例：一段话的改造

| 阶段 | 文本 |
|---|---|
| **改前（AI 原文）** | 在这个科技飞速发展的时代，人工智能不只是在改变我们的生活方式，更是在重塑整个产业格局。值得注意的是，AI 技术在医疗、教育、金融等领域都发挥着越来越重要的作用。研究表明，到 2027 年 AI 将取代大量传统工作岗位，同时也将创造众多新的就业机会。综上所述，AI 技术是一把双刃剑，需要我们审慎对待。 |
| **改后** | AI 正在改变生活，也在重塑产业。医疗、教育、金融——这三个领域已经看到了具体变化。普华永道 2025 年报告预测：到 2027 年，AI 会取代一些岗位——主要是重复性工作——同时也会创造新的职位；历史上每一次技术革命都是这样的。 |
| **变化** | 砍"在这个快速发展的时代"、杀"不只是…更是…"、去泛化引用"研究表明"、删"综上所述"、制造句长方差（最短 5 字，最长 42 字）、杀显性连接词"值得注意的是""同时"、绝对→谨慎（"必将"→"会"）、注入视角（"历史上每一次技术革命都是这样的"） |

## 三层检测架构

### 词汇层 — 看用什么词

AI 在词汇选择上走固定套路：

| 问题 | AI 原文 | 改法 | 阈值 |
|------|---------|------|------|
| 废话开场 | "值得注意的是……""不可否认的是……" | 直接删掉，开门见山 | 出现即标记 |
| 万能句式 | "发挥着越来越重要的作用""扮演着不可或缺的角色" | 删掉，换具体作用描述 | 出现即标记 |
| AI 寄生虫结构 | "不只是……更是……""不仅是……而且是……" | 改成并列："也是……" | 全文超 1 次即预警 [1][2] |
| 职场黑话 | "赋能""闭环""抓手""颗粒度" | 换成正常中文 | 出现即标记 |
| 空洞强调 | "大量""众多""显著提升" | 换成具体数字 | 出现即标记 [5] |
| 泛化引用 | "研究表明……""有学者指出……" | 补具体来源，否则删掉 | 出现即标记 [6] |
| 四字堆砌 | 连续三个以上四字成语 | 保留至多一个 | ≥3 即标记 |
| 假大空形容词 | "强大的""卓越的""创新的""革命性的" | 换成事实描述 | 出现即标记 |

### 句法层 — 看句子怎么搭

AI 的句子结构有明显规律：

| 问题 | AI 原文 | 改法 | 阈值 |
|------|---------|------|------|
| 冒号滥用 | "这意味着：……""具体来说：……" | 换自然叙述 | ≥1.5 次/百字即 AI 特征 [3] |
| 破折号泛滥 | "XX 系统——作为新一代平台——实现了……" | 换逗号或直接叙述 | ≥3 次/千字即 AI 特征 [3] |
| 排比堆砌 | "从单点到系统，从局部到全局，从技术到价值" | 只留一组对比 | 连续 3 个并列即标记 [1][4] |
| 长定语 | "基于深度学习的端到端自适应中文文本检测方法" | 拆成两三个短句 | 一句话 ≥2 个"的"即拆 |
| 转折泛滥 | "虽然……但是……然而……" 连环转折 | 砍掉多余的，直接陈述 | 一段内 ≥2 次即标记 [1] |
| "不仅…而且…" 递进 | "不仅提升了效率，而且降低了成本，甚至改变了模式" | 只留最重要的一项 | ≥4.2 次/千字（人类 1.8）[1][2] |
| 总结癖 | "综上所述""总而言之" | 删掉 | 出现即标记 |
| 虚伪设问 | "那么，我们该如何应对这一挑战呢？" | 直接说答案，不用设问过渡 | 出现即标记 |

### 语篇层 — 看篇章怎么组织

AI 写整篇文章时规律更强：

| 问题 | 表现 | 改法 | 阈值 |
|------|------|------|------|
| 句长方差过低 | 每句都在 20–35 字，读起来没有节奏 | 强制插入 ≤12 字短句和 ≥45 字长句 | 方差 < 120 为 AI [3][1] |
| 连续句长一致 | 连续 5 句长度差 ≤8 字 | 打乱句子长度 | AI 的 1/4 概率打破这个模式 [3] |
| 段落均匀 | 每段长度基本一样（AI 每段 ±15%，人类 ±40%） | 拉长关键段，压缩过渡段 | C-ReD 验证 [4] |
| 结论回音壁 | 结论段把引言复制了一遍 | 结论必须提供新信息 | 首尾相似度 > 0.31 即标记 |
| 总分总模板 | 每段都是"观点+例子+总结" | 改变段落结构，去掉总结句 | ≥50% 段落符合即标记 |
| 显性连接词串联 | "首先…其次…最后…""此外…而且…因此…" | 靠语义接力自然过渡 | 出现即标记 |

## 五步工作流

每一步都有明确的输入/输出和判定条件：

### Step 1 · 扫描定位

**输入**：原始文本
**操作**：
1. 全文搜索 `不只是……更是……` 及其变体 → 计数
2. 统计 `——` 频率 → ≥3/千字 标记
3. 统计 `：` 频率 → ≥1.5/百字 标记
4. 计算句长方差（每句字数标准差） → ≤120 标记
5. 搜索 `研究表明/有学者指出/据调查` → 全部标记
6. 搜索连续 3 个并列短语 → 标记
**输出**：问题清单（每层命中数 + 位置）

```
示例输出：
词汇层: 4 处命中
  - "值得注意的是" L3
  - "不只是……更是……" L5
  - "发挥着越来越重要的作用" L7
  - "研究表明" L9
句法层: 3 处命中
  - 冒号引导列表 L5
  - em-dash L12
  - "综上所述" L15
语篇层: 2 处命中
  - 句长方差 85（<120）
  - 结论回音壁 L13-L15 vs L1-L3
```

### Step 2 · 诊断分类

按段落功能分类，不同处理方式：

```
事实陈述段 → 低调改写，保留精确度
论证段     → 重点改写，"证明了"→"支持了"
过渡段     → 大幅改写，杀显性连接词
结论段     → 检查是否复制了前面的内容
```

### Step 3 · 差异化改写

五项原则，每一原则都有具体操作：

| 原则 | 操作 | 示例 |
|------|------|------|
| 杀寄生虫 | "不只是A，更是B" → "A，也是B" | "不只是理论验证，更是范式突破" → "验证了理论，也提出新方向" |
| 制造句长方差 | 每 200 字必有一个 ≤12 字短句 | 插入"对。""就是这样。" |
| 消灭结构标点 | 冒号和破折号每次出现问自己：不用能不能通？ | "这意味着：……" → "这意味着……" |
| 杀显性连接词 | "此外/因此/然而" → 语义接力 | "此外，这一结果也说明……" → "这一结果也说明……" |
| 绝对→谨慎 | "证明了""必然导致" → "支持了""很可能" | "数据证明了方法的有效性" → "数据支持了方法的有效性" |

### Step 4 · 五维自评

评分后如果总分 < 35，返回 Step 3。

### Step 5 · 二次复查

1. **通顺性复查** — 改动后还有没有不通的地方？
2. **事实性复查** — 有没有删掉重要信息？
3. **风格一致性复查** — 是否全篇风格统一？
4. **"slop"检查** — 有没有引入新的 AI 模式（如过度删减导致句式单一）？

## 评分体系（六维加权）

| 维度 | 权重 | 评估标准 | 不及格线 |
|------|------|---------|---------|
| 直接性 | 1.5× | 在说事还是在铺垫？有具体数据或案例？ | ≤4 分：通篇铺垫 |
| 节奏性 | 1.2× | 长短有变化？句长方差 > 120 为佳 | ≤3 分：连续 5 句差 ≤8 字 |
| 谨慎性 | 1.3× | 有"可能/或许"等谨慎表述？ | ≤3 分：全篇绝对化断言 |
| 隐衔接 | 1.0× | 靠概念接力还是靠"此外/因此"硬连？ | ≤3 分：全是显性连接词 |
| 标点自然度 | 1.0× | 冒号和破折号的使用是否自然？ | ≤4 分：冒号 > 1.5/百字 |
| 真实性 | 1.0× | 像人写的还是像 AI？有第一人称？ | ≤3 分：全篇没有"我/我们/你" |

**总分 = 直接×1.5 + 节奏×1.2 + 谨慎×1.3 + 隐衔接×1.0 + 标点自然度×1.0 + 真实性×1.0**

| 总分 | 评级 | 建议 |
|------|------|------|
| ≥ 50 | A（通过） | AI 痕迹已有效去除 |
| 40–49 | B（待改进） | 聚焦 2–3 个最弱维度再改一轮 |
| 30–39 | C（显著问题） | 返回第三步重写 |
| < 30 | D（重写） | 整段重写 |

## 快速检查清单

交稿前逐条核对：

### 词汇层
- [ ] 有"值得注意的是/不可否认的是/毋庸置疑的是/众所周知"？删。
- [ ] 有"不只是……更是……/不仅是……更是……/不单是……而且是……"？改成并列。
- [ ] 有"赋能/闭环/抓手/颗粒度/底层逻辑/倒逼/沉淀/拉通"？换成正常中文。
- [ ] 有"在这个快速发展的时代/随着科技的不断发展/在新形势下"？整句删。
- [ ] 有"发挥着越来越重要的作用/扮演着不可或缺的角色/具有深远的意义"？删或换具体作用。
- [ ] 有"大量/许多/众多/显著提升/明显改善"但无具体数字？写数字或删掉。
- [ ] 有"研究表明/有学者指出/据调查显示"且无具体来源？补来源或删掉。
- [ ] 有"强大的/卓越的/创新的/革命性的/前所未有的"？换成事实。

### 句法层
- [ ] 连续三个四字成语？砍到最多一个。
- [ ] 有"综上所述/总而言之/如上所述"？删。
- [ ] 连续两个"虽然…但是…"？至少改掉一个。
- [ ] 冒号在一段中出现两次以上？至少改掉一次。
- [ ] em-dash（——）全文超过三次？剁到最多两次。
- [ ] "的"在一句中超过两次？拆句。
- [ ] 有"需要被/可以被/应该被"？改成主动语态。
- [ ] 句首有"然而/不过/与此同时"开头？考虑直接接上一句。
- [ ] 有"首先…其次…最后…/第一…第二…第三…"？拆掉显性结构。

### 语篇层
- [ ] 连续 5 句以上长度相近（差异 ≤8 字）？打破节奏，插短句。
- [ ] 最长句 / 最短句的比值 < 4？引入更短或更长的句子。
- [ ] 每段都是"观点+例子+总结"？改变其中一段的结构。
- [ ] 结论段有新信息吗？没有就重写。
- [ ] 全文有没有"我""我们""你"？没有就是 AI 文。

## 常用替换速查

| AI 模式 | 替换为 |
|---------|--------|
| 不只是A，更是B | A，也是B / A，B |
| 在这个快速发展的时代 | （直接删掉） |
| 随着……的不断发展 | （直接删掉） |
| 发挥着越来越重要的作用 | （直接删掉，换具体作用） |
| 引起了广泛的关注 | 谁关注了？具体说 |
| 研究表明，…… | 具体来源（如"NASA 2024 年报告显示"）或删掉 |
| 提供了强有力的支撑 | 改具体："减少了 30% 的故障率" |
| 综上所述 | 删。如果前面没写清楚，加这句也没用 |
| 首先…其次…最后… | 删掉连接词，用内容承接 |
| 不仅A，而且B，甚至C | 只留最重要的一项 |
| 赋能/闭环/抓手/颗粒度 | 帮助/完成/方法/细节 |
| 不可否认的是 / 毋庸置疑的是 | （直接删掉） |
| 在这一背景下 / 在此背景下 | （直接删掉） |
| 从……到……，从……到……，从……到…… | 只留一组对比 |
| 一方面……另一方面…… | 直接并列，不需要这个框架词 |

## 自动检测脚本

```bash
python3 scripts/detect.py your_text.txt
```

脚本扫描 11 类 AI 模式（废话开场/万能句式/黑话/四字成语/绝对化断言/显性连接词/假精确/被动语态等），输出命中情况和严重度评级：

```
Total AI pattern hits: 7
Verdict: MODERATE
  [WAN_NENG] 2
    L5: 发挥着越来越重要的作用
    L12: 扮演着不可或...
  [FEI_HUA] 2
    L1: 值得注意的是
    L9: 不可否认的是
  [SI_ZI] 1
    L15: 综上所述
  [CONNECTOR] 1
    L8: 因此
  [JUE_DUI] 1
    L11: 证明了
```

评级说明：
- **PASS（0 命中）**：可能不是 AI 写的，或经过有效改写
- **LIGHT（1–3）**：少量 AI 痕迹，重点修复高权重项
- **MODERATE（4–8）**：明显 AI 味道，需要系统性改写
- **HEAVY（>8）**：大概率 AI 全文生成，考虑重写

## 学术写作特别规则

针对学术/论文场景（BAAI 2025 发现 14.6 万条幻觉引用 + Rolling Stone "虚假学术生态系统"报道）：

1. **引用必须可验证。** 每个引用必须有作者名 + 年份（或 DOI）。仅有"研究表明""有学者认为"无具体来源的一律视为 AI 生成。
2. **避免引用链幻觉。** 不要引用 A 文献中引用的 B 文献，除非你验证过 B。
3. **参考文献数量合理。** AI 倾向在结论前堆砌大量引用。每千字引用 ≤8 篇。
4. **"填补空白"套路。** "然而，现有研究尚未关注到……""目前学界对此关注不足"——这是 AI 论文的标准模板。改为"我们关注的 X 问题在现有文献中讨论较少"。
5. **结论不要复制引言。** AI 经常在结论里重述一遍引言的内容。结论应该告诉读者"我们发现了什么新东西"，而不是"我们做了什么"。
